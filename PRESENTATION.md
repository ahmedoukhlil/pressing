## 1. لوحة التحكم

**المسار:** `/dashboard`

ملخص يومي فوري عند فتح التطبيق يعرض:

| البطاقة | الوصف |
|--------|-------|
| **المبيعات المحصّلة** | مجموع المبالغ المدفوعة فعلياً في تاريخ محدد |
| **مستحقات غير محصّلة** | مجموع المبالغ المتبقية على الزبناء في تاريخ محدد |
| **طلبات جاهزة** | عدد الطلبات في حالة "جاهز" لم تُسلَّم بعد |
| **طلبات متأخرة** | طلبات تجاوزت تاريخ التسليم المتوقع |

### جدول الإيرادات اليومية
- فلترة بالتاريخ وطريقة الدفع
- يعرض: رقم الطلبية، الزبون، المبلغ، طريقة الدفع، التاريخ
- الأحدث يظهر أولاً
- تصفح بالصفحات (Pagination)

## 2. إيداع الملابس — نقطة البيع

**المسار:** `/pos`

الواجهة الرئيسية لاستقبال ملابس الزبناء وإنشاء طلبيات جديدة.

### الوظائف الرئيسية

**البحث عن الزبون:**
- بحث بالاسم، رقم الهاتف، أو الرمز
- إنشاء زبون جديد مباشرة من الشاشة

**سلة الطلب:**
- إضافة خدمات من قائمة الخدمات المتاحة
- تحديد الكمية لكل خدمة
- حذف أي خدمة من السلة

**الخصومات:**
- خصم عند الإيداع (نسبة مئوية أو مبلغ ثابت)
- خصم عند التسوية (مبلغ ثابت)

**الدفع:**
- دفع جزئي أو كلي عند الإيداع
- اختيار طريقة الدفع (نقداً، BANKILY، SADAD...)
- المبلغ المتبقي يُتابع تلقائياً

**الإيصال:**
- طباعة تذكرة فورياً بعد تسجيل الطلب
- التذكرة تتضمن: رقم الطلبية، تفاصيل الخدمات، الأسعار، المبلغ المدفوع والمتبقي، ختم المسؤولية

**تحديد موعد التسليم:**
- تاريخ ووقت التسليم المتوقع

## 3. الطلبات والبحث

**المسار:** `/recherche`

إدارة كاملة لجميع الطلبيات.

### البحث والفلترة
- بحث نصي: اسم الزبون، رقم الهاتف، رقم الطلبية
- فلترة بالحالة: قيد المعالجة / جاهز / تم التسليم / ملغى
- فلترة بتاريخ الإيداع (من - إلى)
- تصفح بالصفحات

### حالات الطلبية

| الحالة | المعنى |
|--------|--------|
| **قيد المعالجة** | الملابس في المغسلة |
| **جاهز** | الملابس جاهزة للاستلام |
| **تم التسليم** | سُلِّمت للزبون |
| **ملغى** | طلبية ملغاة |

### إجراءات على الطلبية
- تغيير الحالة
- تسجيل دفعة جديدة (مع طريقة الدفع)
- طباعة التذكرة
- عرض تفاصيل الخدمات والمدفوعات

### تصدير نتائج البحث
- تصدير PDF يشمل: معلومات الزبون، تفاصيل كل خدمة، الكميات، الأسعار، المدفوع والمتبقي، الملخص الإجمالي

## 4. الزبناء

### قائمة الزبناء
**المسار:** `/clients`

- بحث بالاسم، الهاتف، أو الرمز
- عرض الرمز المتسلسل لكل زبون (0001، 0002...)
- تصفح بالصفحات

### إنشاء / تعديل زبون
**المسار:** `/clients/nouveau` · `/clients/{id}`

الحقول: الاسم الكامل، رقم الهاتف، البريد الإلكتروني، العنوان.

### الزبناء المدينون
**المسار:** `/clients/endettes`

قائمة مخصصة تعرض فقط الزبناء الذين عليهم مبالغ غير مسددة، مرتبة حسب المبلغ المتبقي.

### حساب الزبون
**المسار:** `/clients/{id}/compte`

صفحة شاملة تعرض:

**الملخص المالي:**
- إجمالي الطلبات (المفوتر الكلي)
- إجمالي المدفوع
- ما على الزبون (متبقي)
- ما على المغسلة (في حال الدفع الزائد)

**لائحة الطلبيات:**
- جميع الطلبيات مرتبة من الأحدث
- تفاصيل كل طلبية: الخدمات، الكميات، الأسعار
- **المدفوعات لكل طلبية:** كل دفعة مع طريقة الدفع والتاريخ

## 5. المصروفات

**المسار:** `/depenses`

### الوظائف
- تسجيل مصروف جديد: نوع المصروف، المبلغ، التاريخ والوقت، طريقة الدفع، المورد، المرجع، ملاحظات
- فلترة: بالنوع، التاريخ، طريقة الدفع، المورد، الحالة
- حالات المصروف: مصادق عليه / في الانتظار

### أنواع المصروفات
**المسار:** `/parametrage/types-depenses`

تصنيف مخصص للمصروفات (أجور، مواد، خدمات...) مع أيقونة ولون لكل نوع.

## 6. الإيرادات والمصروفات

**المسار:** `/finances/recettes-depenses`

تقرير مالي متكامل بأربعة أوضاع عرض:

### أوضاع التجميع

| الوضع | الوصف |
|-------|-------|
| **يومي** | كل يوم في الشهر المحدد |
| **أسبوعي** | أسابيع الشهر المحدد |
| **شهري** | أشهر السنة المحددة |
| **سنوي** | كل السنوات في قاعدة البيانات |

### البطاقات الإحصائية
- **إجمالي الإيرادات** — مبالغ محصّلة فعلياً من الزبناء
- **إجمالي المصروفات** — مصروفات مصادق عليها في الفترة
- **صافي الربح** — إيرادات ناقص مصروفات
- **مستحقات غير محصّلة** — مبالغ متبقية على طلبيات الفترة

### الإيرادات حسب طريقة الدفع
- بطاقات قابلة للنقر لكل طريقة دفع
- شريط تقدم يوضح نسبة كل وسيلة
- عند النقر: نافذة تعرض تفاصيل كل عملية (إيرادات ومصروفات) لتلك الطريقة

### جدول الفترات
الأعمدة: الفترة، الإيرادات، المصروفات، الصافي، المستحقات، مقارنة بصرية، حالة (ربح/خسارة)

### الكشف التفصيلي للعمليات
- كل عملية منفردة (إيراد أو مصروف) مع: التاريخ، النوع، البيان، طريقة الدفع، المبلغ
- **مُقسَّم إلى صفحات** (25 عملية في الصفحة)
- تصدير PDF أو Excel

### التصدير
- **PDF:** تقرير مُنسَّق بالعربية مع المجاميع
- **Excel (.xls):** ملف SpreadsheetML يُفتح في Excel مع دعم RTL والأرقام الصحيحة

## 7. الإعدادات

### الخدمات
**المسار:** `/services`

إدارة قائمة الخدمات المقدمة في المغسلة:
- إنشاء/تعديل/حذف خدمة
- الحقول: الاسم (عربي/فرنسي)، السعر، الأيقونة، الصورة، الترتيب، الحالة (نشط/غير نشط)

### ترتيب الزبناء (CA Ranking)
**المسار:** `/parametrage/fidelite`

تصنيف الزبناء حسب رقم الأعمال:
- **Top 10 زبناء** حسب مجموع المبالغ المحصّلة فعلياً
- مقارنة بين الشهر الحالي والشهر السابق
- مؤشر التطور (▲ زيادة / ▼ انخفاض)
- فلترة بالسنة والشهر

### طرق الدفع
**المسار:** `/parametrage/modes-paiement`

- إضافة/تعديل طرق الدفع (نقداً، BANKILY، SADAD، تحويل بنكي...)
- الحقول: الاسم، الرمز، الأيقونة، الترتيب، الحالة

### الموظفون
**المسار:** `/parametrage/employes`

إدارة كاملة للموظفين:
- **بيانات الموظف:** الاسم، الهاتف، المنصب، تاريخ التوظيف، الراتب الإجمالي
- **وثائق الهوية:** صورة رقم التعريف (وجهان)
- **السلف:** تسجيل السلفة، المبلغ، السبب، وتتبع الخصم
- **صرف الراتب:** احتساب الراتب الصافي (إجمالي − السلفة)، تسجيل الصرف كمصروف تلقائياً
- **حذف الموظف:** مع تأكيد ومسح السجلات المرتبطة

### المخزون
**المسار:** `/parametrage/stock-consommables`

- إدارة المواد الاستهلاكية (صابون، مواد كيميائية...)
- تسجيل حركات الدخول/الخروج
- تنبيه عند تجاوز حد الانذار

### الموردون
**المسار:** `/parametrage/fournisseurs`

- اسم المورد، الهاتف، رقم التعريف الجبائي (NIF)
- ربط المصروفات بالمورد

## 8. الإدارة

### الفروع (Succursales)
**المسار:** `/admin/succursales`

- إنشاء وإدارة فروع المغسلة
- كل فرع له بياناته المستقلة
- التبديل بين الفروع من القائمة العلوية (مدير المغسلة)

### المستخدمون
**المسار:** `/admin/utilisateurs`

- إنشاء حسابات للموظفين والمديرين
- تعيين الدور (Role) لكل مستخدم
- ربط المستخدم بالفرع

### الأدوار والصلاحيات
**المسار:** `/admin/roles`

- إنشاء أدوار مخصصة (مدير، موظف استقبال، محاسب...)
- تعيين صلاحيات محددة لكل دور
- التحكم في الوصول إلى كل صفحة وكل إجراء

## 9. التصدير والطباعة

### تذكرة الطلبية (Ticket)
**المسار:** `/commandes/{id}/ticket`

طباعة فورية عند الإيداع أو الطلب:
- اسم المغسلة، الهاتف، العنوان
- رقم الطلبية وتاريخ الإيداع
- معلومات الزبون
- تفاصيل الخدمات والكميات والأسعار
- الملخص المالي (الإجمالي، المدفوع، المتبقي)
- **تنبيه:** "المغسلة غير مسؤولة عن الملابس المودعة منذ أكثر من 7 أيام"
- رسالة مخصصة في القدم (من الإعدادات)

### تصدير الطلبيات PDF
- قائمة الطلبيات مع فلاتر البحث
- تفاصيل كل طلبية: الخدمات، الكميات، الأسعار، المدفوع والمتبقي
- الملخص الإجمالي (إجمالي القطع، المفوتر، المحصّل، غير المحصّل)

### تصدير المصروفات PDF
- قائمة كاملة للمصروفات مع جميع التفاصيل

### تصدير المخزون PDF
- حالة المخزون الحالية مع مستويات التنبيه

### تصدير التفاصيل المالية
| الصيغة | التفاصيل |
|--------|---------|
| **PDF** | تقرير مُنسَّق بالعربية، RTL، خط DejaVu |
| **Excel (.xls)** | SpreadsheetML مع دعم RTL، أرقام صحيحة، ترويسات بخط عريض |

## 10. الصلاحيات والأدوار

النظام يعتمد على **Spatie Laravel Permission** لإدارة الأدوار والصلاحيات.

### نموذج الصلاحيات

كل صفحة وكل إجراء محمي بصلاحية منفصلة، مثل:

| الصلاحية | الوصف |
|---------|-------|
| `view.dashboard` | عرض لوحة التحكم |
| `view.pos` | الوصول لنقطة البيع |
| `view.clients.index` | عرض وإدارة الزبناء |
| `view.depenses.index` | عرض المصروفات |
| `view.finances.recettes-depenses` | عرض التقارير المالية |
| `export.commandes.pdf` | تصدير الطلبيات PDF |
| `view.admin.users.index` | إدارة المستخدمين والأدوار |
| `view.parametrage.*` | الوصول لصفحات الإعدادات |

## 11. البنية التقنية

### المكدس التقني (Tech Stack)

| الطبقة | التقنية |
|--------|--------|
| **Framework** | Laravel 12 |
| **Frontend** | Livewire 3 (Reactive, بدون إعادة تحميل الصفحة) |
| **CSS** | Tailwind CSS |
| **PDF** | mPDF 8 (دعم كامل للعربية وRTL) |
| **Excel** | SpreadsheetML (XML Excel 2003، بدون مكتبة خارجية) |
| **أيقونات** | Flaticon Uicons (fi-rr-*) |
| **الصلاحيات** | Spatie Laravel Permission |
| **قاعدة البيانات** | MySQL (عبر XAMPP) |

### النماذج الرئيسية (Models)

```
Commande          — الطلبيات
DetailCommande    — تفاصيل الطلبيات (خط بخدمة)
Service           — الخدمات
Client            — الزبناء
CaisseOperation   — عمليات الصندوق (إيرادات)
Depense           — المصروفات
TypeDepense       — أنواع المصروفات
ModePaiement      — طرق الدفع
Employe           — الموظفون
AvanceSalaire     — السلف
Consommable       — المواد الاستهلاكية
StockMouvement    — حركات المخزون
Fournisseur       — الموردون
Succursale        — الفروع
User              — المستخدمون
Setting           — إعدادات النظام
```

### الميزات المعمارية
- **تعدد الفروع:** كل سجل مرتبط بفرع (Succursale) عبر `forCurrentSuccursale()` scope
- **Livewire Computed Properties:** الحسابات كـ computed properties مُخزَّنة مؤقتاً
- **Pagination مزدوجة:** `WithPagination` مع صفحات مسماة للتعايش في نفس الصفحة
- **RTL كامل:** كل الواجهات، الجداول، والمستندات المُصدَّرة بالعربية من اليمين لليسار
